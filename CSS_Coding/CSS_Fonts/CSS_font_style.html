CSS Fonts Style

The font-styleproperty is mostly used to specify italic text.

This property has three values:

  • normal - The text is shown normally
  • italic - The text is shown in italic
  • oblique - The text is "leaning" (oblique is very similiar to italic, but less supported)

CSS Font Variant

The font-variant property specifies whether or not a text should be displayed in a small-caps font.

In a small-caps font, all lowercase letters re convertes to uppercase letters. However , the converted uppercase letters appears in a smaller font size than the original uppercase letters in the text.
