CSS Basic Font Family

Serif fonts have a small stroke at the edges of each letter. They create a sense of formality and elegence.

sans-serif fonts have clean lines (no small strokes attached).They create a modern and minimalistic look.

Momospace fonts- here all the letters have the same fixed width. They create a mechanical look.

Cursive fonts imitate human handwriting.

Fantasy fonts are decorative/playful fonts.

'Times New Roman', Times, serif;
