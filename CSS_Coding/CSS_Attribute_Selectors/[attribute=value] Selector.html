CSS Attribute Selectors

CSS [attribute|="value"] Selector

The [attribute|="value"] selector is used to select elements with a attribute value, whose value can be exactly the specified value, or the specified value followed by a hyphen (-).

Note: The value has to be a whole word, either alone, like class="top", or followed by a hyphen( - ), like class="top-text".
