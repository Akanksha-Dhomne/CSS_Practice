CSS Text Decoration

We can do text Decoration by using following properties:

  • text-decoration-line
  • text-decoration-color
  • text-decoration-style
  • text-decoration-thickness
  • text-decoration

Add a Decoration line to text

The text-decoration-line property is used to add a Decoration line to text.

Tip:We can combine more than one value, like bidi-overline and underline to display lines both over and under a text.
